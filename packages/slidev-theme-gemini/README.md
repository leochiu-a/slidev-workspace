# slidev-theme-gemini

A Slidev theme inspired by a cinematic cover layout.

## Usage

```md
---
theme: gemini
layout: cover
tag: Gemini Session
titleLine1: AI Momentum
titleLine2: Where JavaScript Fits
subtitle: |
  Reframing the role of JS builders in the AI era.
coverImage: /your-cover.jpg
hint: to begin
---
```
